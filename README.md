# Project polishpicker
I'm currently working on a project for keeping track of my collection of nail polishes. One important feature will be a "polish picker tool" inspired by the outfir generator in Clueless. 

[Check out the current version here](https://polishpicker.vercel.app/)

- [Sanity](https://www.sanity.io/)
- [Node](https://nodejs.org/en/)
